add(n1, n2, ...) ​

The add function adds together two or more numbers.

js
import { add } from '@abw/badger-maths'

add(1.2, 0.1);            // 1.2
add(2.2, 2.2, 2.2)        // 6.6
add(-3.14, -3.14, -3.14)  // -9.42
